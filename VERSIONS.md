ΒΙΒΛΙΟ ΕΚΡΗΚΤΙΚΩΝ ΥΛΩΝ (EXPVAULT) — ΙΣΤΟΡΙΚΟ ΕΚΔΟΣΕΩΝ
Τελευταία ενημέρωση: 2026-07-10

Τρέχουσα έκδοση: v1.0.4 (GitHub release: https://github.com/papadcha/expvault/releases/tag/v1.0.4)
====================================================


v1.0.4 — 2026-07-10
----------------------------------------------------------------
Νέο: Άδεια ανά νόμιμη κατηγορία εκρηκτικού αντί ανά υλικό — οι
εγκεκριμένες ποσότητες μιας άδειας καταχωρούνται πλέον ανά κατηγορία
(π.χ. Ζελατοδυναμίτιδα, Καψύλλια NONEL) και όχι ανά συγκεκριμένο υλικό.
- Διαχωρισμός νόμιμης κατηγορίας καψυλλίων στο Δελτίο Δραστηριότητας.
- Ρυθμίσεις backup (backup_config.json).
Fix: διόρθωση σειράς migrations στη βάση — προκαλούσε crash σε νέες
εγκαταστάσεις.
Fix: escaping μη έμπιστου κειμένου στο UI (ονόματα υλικών/προμηθευτών/
αδειών, παρατηρήσεις, μηνύματα σφάλματος) — πρόληψη έγχυσης HTML/JS από
μη έμπιστο περιεχόμενο (π.χ. Εισαγωγή PDF), σκλήρυνση update-checker.


v1.0.3 — 2026-06-27
----------------------------------------------------------------
Νέο: Windows installer με PyInstaller + electron-builder — πλήρες
πακέτο εγκατάστασης χωρίς εξωτερικές εξαρτήσεις (Python/Node).
Νέο: έλεγχος και αυτόματη ενημέρωση έκδοσης κατά την εκκίνηση.
Fix: σωστή εμφάνιση S3-compatible remotes (Mega, Wasabi, κλπ) στη λίστα
backup.
Fix: εντοπισμός rclone.conf στο σωστό path για Windows.


v1.0.2 — 2026-06-27
----------------------------------------------------------------
Νέο: εξαγωγή/εισαγωγή προτύπων παραστατικών με έλεγχο διπλοτύπου —
δυνατότητα κοινοποίησης εκπαιδευμένων προτύπων PDF parser μεταξύ
εγκαταστάσεων.


v1.0.1 — 2026-06-27
----------------------------------------------------------------
Νέο: bundle γραμματοσειρών (Iosevka Regular/Bold, JetBrains Mono) για
βέλτιστη εμφάνιση σε κάθε PDF export, ανεξάρτητα από τις γραμματοσειρές
του συστήματος.
Docs: τεκμηρίωση εγκατάστασης rclone για Windows στον Οδηγό Χρήσης.


v1.0.0 — 2026-06-27
----------------------------------------------------------------
Πρώτη έκδοση του Βιβλίου Εκρηκτικών Υλών: καταχώρηση κινήσεων
(αγορές/αναλώσεις), αποθέματα, παραστατικά, εισαγωγή/parsing PDF,
υπολογισμός συμφωνίας, άδειες αγοράς, είδη εκρηκτικών, προμηθευτές,
backup σε τοπικό φάκελο / cloud remote (rclone).
