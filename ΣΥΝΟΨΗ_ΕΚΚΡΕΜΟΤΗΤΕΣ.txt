ΤΙ ΕΙΝΑΙ
─────────
Electron desktop εφαρμογή για Βιβλίο Αγοράς και Κατανάλωσης
Εκρηκτικών Υλών (λατομείο). Τρέχει τοπικά, SQLite βάση,
Python backend για exports.


ΤΙ ΕΧΕΙ ΥΛΟΠΟΙΗΘΕΙ
────────────────────
✅  Καταχώρηση κινήσεων (Εισαγωγή / Επιστροφή / Κατανάλωση)
✅  Import παραστατικών από PDF
✅  Υπολογιστής
        Σενάριο 1: Αγορά + Κατανάλωση → Επιστροφή
        Σενάριο 2: Αγορά + Επιστροφή  → Κατανάλωση
✅  Καταχώρηση επιστροφής απευθείας από Υπολογιστή
✅  Edit κινήσεων — modal με αγορά + επιστροφή + κατανάλωση μαζί
✅  Σελίδα Παραστατικά
        · Μαζική ανάθεση αρ.παραστατικού σε εκκρεμείς επιστροφές
        · Μετονομασία / διόρθωση υπάρχοντος παραστατικού
✅  Export PDF
        Σελ.1: Αγορές / Επιστροφές
        Σελ.2: Καταναλώσεις
        (landscape A4, 4 modes NONEL: detail/subgroup/grouped/total)
✅  Export Excel
        4 φύλλα: Αναλυτικό / Ανά καθυστέρηση / Ανά τύπο NONEL / Σύνολο
✅  Export Word (.docx)
        Σελ.1: non-NONEL υλικά (landscape A4)
        Σελ.2: NONEL υλικά (landscape A4)
✅  Διαχείριση Υλικών, Προμηθευτών, Αδειών
✅  Αποθέματα (υπόλοιπα ανά υλικό)
